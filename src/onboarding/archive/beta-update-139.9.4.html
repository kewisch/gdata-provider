What's New in 139.9.4

Features

  • Don't set PUBLIC visibility on events received through invitations

Fixes

  • Fix a console error where conferenceSolutions was not defined
  • Correctly serialize snoozes on recurring events
  • Work around organizer participation status not being set correctly
  • Don't remember older snoozes on recurring events to avoid property size limitations
  • Better allow accepting events with Windows timezone names

If there are issues, the most likely would be around snoozing recurring events or not being able to set your participation status as an event organizer. I'm hoping though the experience is smoother than before, as the patches fix a few edge cases that have been reported.

I haven't received any feedback on the edit conferencing feature from 139.9.3. No news is good news? Please use the release discussion if you have feedback in either direction.
