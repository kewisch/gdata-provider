Thank you for being a beta tester for the Provider for Google Calendar!

What's New in 139.9.2

Please re-subscribe to your calendars with this update, some calendar properties were incorrectly set.

Features

  • Read-only support for attachments
  • Sort primary and selected calendars first in the new calendar wizard
  • Show color dots in the new calendar dialog and use the calendar color
  • Do exponential backoff on all requests
  • Include email in calendar name if duplicate to an existing calendar
  • Add input validation in session screen

Fixes

  • Invitations can be accepted again (fix missing iCalUID error)
  • No longer send a client side invitation accept message, let the server handle it
  • Use summaryOverride calendar name if specified
  • Set correct colorId from API response
  • Don't constantly show the warning icon
  • Make it possible to complete tasks again
  • Fix a console message that showed "undefined" instead of a title
  • No longer break for events with differently formatted LASTACK and SNOOZE-TIME
  • Improve OAuth2 error handling

A big thank you to everyone who has already provided feedback or filed bugs. Special thanks to jonschz who also sent a pull request!
